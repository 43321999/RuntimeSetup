# « Quick Create » ubuntu on hyperv
- Select «Ubuntu 22.04», reName, change network and click «Create VM»;
- check «don't send system info» and Done.

## local guest shell
```sh
sudo apt update
sudo apt install -y openssh-server

```
~~```service-ssh-start```~~
## local host shell
```sh
ssh-copy-id username@remote_host
```
